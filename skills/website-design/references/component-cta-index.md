# Component CTA — Index

**CTA section philosophy:** Conversion sections should feel inevitable, not desperate. The best CTA is a logical conclusion to the content that precedes it. Match CTA intensity to business type and dimensional signal.

Every page has exactly one primary CTA section and zero or one secondary CTA elements.

Selection logic:
1. Product/SaaS self-serve signup? → CTABanner (accent or glass variant)
2. Premium service consultation? → ContactGateway or CTABanner (dark variant)
3. Philosophy/differentiation statement? → Manifesto (placed mid-page, not as final CTA)
4. Content-first newsletter conversion? → NewsletterCapture

## Comparison Table

| Component | Use When | Visual Weight | Motion Profile | Dimension Fit |
|---|---|---|---|---|
| CTABanner | Universal primary CTA; three visual variants | medium | minimal | all dimensions: high (universal) |
| Manifesto | Philosophy is the differentiator; contrast + power statement | heavy | moderate | contrast-dark: high, energy-energetic: high, energy-restrained + editorial: high |
| NewsletterCapture | Content-first business; newsletter is primary conversion | light | none | contrast-light + editorial: high, energy-restrained: medium |
| ContactGateway | Service business consultation CTA; calm authority | light | none | contrast-dark: high, contrast-light: high, energy-restrained: high |
